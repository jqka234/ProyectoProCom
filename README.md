El unico requisito para correrlo es correrlo desde el archhivo main, tiene que estar en un proytecto.
Es una aplicación en Java que permite buscar un medicamento y le va a entregar las causas principales y los efectos secundarios.
El programa va a obtener sus efectos secundarios desde la API de la FDA y mostrarlos traducidos al español usando otra API, todo mediante JOptionPane y aplicando principios de POO.
